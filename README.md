A full-stack marketplace application built with React, Node.js, and MongoDB. Features separate admin, client, and API interfaces.

## Structure

```
EmsMarketplace/
├── admin/     # Admin dashboard
├── client/    # Client-facing app
└── api/       # Backend API server
```

## Setup

Each directory (admin, client, api) needs its dependencies installed:

```bash
cd <directory>
npm install --legacy-peer-deps
npm start
```

Run all three simultaneously for the full experience.

- **Admin**: Admin management interface
- **Client**: http://localhost:3000
- **API**: Backend server

## Tech Stack

- **Frontend**: React, Firebase
- **Backend**: Node.js
- **Database**: MongoDB
